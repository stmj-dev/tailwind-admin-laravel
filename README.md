# LARAVEL ADMIN

A tailwind admin template using tailwind css and flowbite. including components (easy to customize) and responsive design

## COMPONENTS AVAILABLE

-   Breadcumb
-   Breadcrumb Link
-   Button
-   Card
-   Dark Mode Toggle
-   Input
-   Input Checkbod
-   Input File
-   Input Radio
-   Input Toggle
-   Modal
-   Modal Delete
-   Table
-   Toast / Notification
-   Validate Error / Alerts
